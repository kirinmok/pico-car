// MISSION 01 of 03 — 電壓判讀
這顆電池
還能飛嗎？
鋰電池的電壓就像它的血量計。電壓太低，電池會永久損壞，再也充不回來。
先看這份電壓對照表：
電壓狀態意思
4.35 V🟢 滿電可以起飛
3.7 V 🟡 正常繼續飛，沒問題

3.0 V 🔴 截止立刻停止放電，否則永久損壞
2.8 V 💀 危險已超過截止，電池內部受損

// MISSION 02 of 03 — 異常電池處置
這顆電池
外觀不對勁
鋰電池膨脹（鼓包）是危險警告。
原因：過充、過放或老化導致內部產生氣體。
膨脹電池隨時可能起火或爆炸，必須立即處置。
❌ 絕對不能用水冷卻鋰電池——水會讓鋰電池反應更劇烈，更危險。

// MISSION 03 of 03 — 電池規格計算
這顆電池
夠力嗎？
C 數代表電池的放電速率。
用 C 數可以算出電池最大能輸出多少電流。
電流不夠 → 馬達無力 → 無人機墜機。
最大放電電流（mA）= 容量（mAh）× C 數
// 任務情境 第三顆電池標示：450 mAh，30C
無人機四顆馬達全速需要：約 3,500 mA

請算出這顆電池的最大放電電流。
